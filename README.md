# MERN Website

This project is structured as a MERN stack application.

## Structure
- `/frontend`: Contains the React/Vite-based frontend.
- `/backend`: Contains the Node/Express backend API.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start` or `node server.js`

### Frontend
1. `cd frontend`
2. Open `index.html` in a live server (currently using CDN-based React).
